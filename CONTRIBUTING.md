# Contributing to Claude Orchestra

Thanks for your interest in contributing! We welcome PRs from everyone.

## Quick Start

```bash
# Fork and clone the repo, then:
pip install -e ".[dev]"

# Run tests
pytest

# Make your changes, run tests again, submit PR
```

## What We Need Help With

- **Session analytics features** (v0.2 roadmap)
- **Dashboard improvements**
- **Documentation**
- **Testing on different platforms**

## Code Style

- **Black** for formatting
- **Ruff** for linting
- Type hints preferred

Run before submitting:
```bash
black .
ruff check .
```

## PR Guidelines

- Keep PRs small and focused
- Add tests for new features
- Update docs if needed

## Community

- Use **GitHub Issues** for bugs and feature requests
- Be kind

---

Questions? Open an issue. We're happy to help!
